# Power BI Projects

A small collection of Power BI dashboards built around real business and operational questions. Each project folder includes the `.pbix` file and a detailed README with the full analysis.

## Explore the Projects

- [Coffee Sales](./Coffee%20Sales/)  
  Coffee shop sales analysis focused on revenue drivers, pricing, product performance, and daily demand patterns.
- [Hospital ER](./Hospital%20ER/)  
  Emergency room analytics covering patient flow, waiting time, referrals, and overall service performance.
- [Pizza Sales](./Pizza%20Sales/)  
  Pizza sales dashboard exploring revenue trends, customer ordering behavior, menu performance, and ingredient efficiency.

Open any folder to read more about the project details.
