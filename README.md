# Power BI Case Study Portfolio

Recruiters and hiring managers sometimes land on GitHub before they visit a portfolio site. This repository is designed to make that experience useful on its own, with each project presented as a visual case study that combines business context, dashboard storytelling, and direct access to the live Power BI report.

> GitHub README files do not support live Power BI iframes. Each project below includes a linked dashboard preview that opens the interactive report in Power BI.
